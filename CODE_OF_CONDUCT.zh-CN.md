# 行为准则（Code of Conduct）

> English version: `CODE_OF_CONDUCT.md`

本项目遵循 **Contributor Covenant Code of Conduct**。

如中英文版本存在差异，以英文版本为准。

## 我们的承诺（Our pledge）

我们承诺为所有人提供一个无骚扰的社区参与体验，不论年龄、体型、可见或不可见的残障、族裔、性特征、性别认同与表达、经验水平、教育背景、社会经济状况、国籍、外貌、种族、宗教，或性取向。

## 我们的标准（Our standards）

有助于营造积极环境的行为示例包括：

- 对他人表现出同理心与善意
- 尊重不同观点、意见与经验
- 提供并友好接受建设性反馈
- 为自己的错误承担责任，并向受影响者道歉
- 关注对社区最有益的事情

不可接受行为示例包括：

- 使用性暗示语言或图像，以及不受欢迎的性关注或性挑逗
- 挑衅、侮辱或贬损性评论，以及针对个人或政治立场的攻击
- 公开或私下骚扰
- 未经明确许可发布他人隐私信息（例如真实地址或邮箱）
- 其他在专业场景中可被合理认为不恰当的行为

## 执行责任（Enforcement responsibilities）

社区领导者负责澄清并执行可接受行为标准，并会对其认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

## 适用范围（Scope）

本准则适用于所有社区空间；当个人以官方身份代表社区在公共场合活动时，也同样适用。

## 执行（Enforcement）

如发生辱骂、骚扰或其他不可接受行为，可通过 GitHub 的举报与管理工具进行报告，包括向仓库维护者提交报告或使用 GitHub 内置的举报流程。

## 致谢（Attribution）
