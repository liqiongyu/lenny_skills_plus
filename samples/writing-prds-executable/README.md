# Writing PRDs (Executable)

This skill helps an agent draft or improve:
- PR/FAQ
- PRD / product spec
- AI eval specs (LLM-as-judge)
- Prompt sets / prototype prompts

## Install

Copy this folder into your project:

`.claude/skills/writing-prds-executable/`

## Invoke

Examples:
- “Help me write a PRD for <feature>.”
- “Turn these notes into a PR/FAQ then a lean PRD.”
- “For this AI feature, draft an eval spec and a prompt set.”

## Templates & quality

Templates live in `assets/`.
Quality checks live in `references/`.
