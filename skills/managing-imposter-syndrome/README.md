# Managing Imposter Syndrome (Skill Pack)

Use this skill to turn “I feel like a fraud” into an **executable Imposter Syndrome Management Pack**: trigger map, reframes, evidence bank, a 2–4 week experiment plan, support scripts, and a maintenance routine.

## How to invoke

Example prompts:
- “Use `managing-imposter-syndrome` to create an Imposter Syndrome Management Pack. Here are 2 recent situations where I felt like a fraud.”
- “Use `managing-imposter-syndrome` to build an exposure ladder + feedback plan for my new role.”

## What you get

The skill outputs an **Imposter Syndrome Management Pack** (in chat as Markdown; or as files if requested). Templates, checklists, and expanded guidance live in `references/`.
