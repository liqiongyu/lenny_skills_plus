# Onboarding New Hires (Skill Pack)

Design a high-signal onboarding experience and produce a **New Hire Onboarding Pack** (preboarding + first-week plan + listening tour + working agreement + 30/60/90 plan).

## Install

- Canonical location: `skills/onboarding-new-hires/`
- For auto-discovery, mirror to:
  - `.codex/skills/onboarding-new-hires/`
  - `.claude/skills/onboarding-new-hires/`

## Use

Invoke this skill when you need a concrete onboarding plan and artifacts:

- “Create an onboarding pack for a new Senior PM joining on <date>.”
- “Design a listening tour and 30-day state-of-the-union memo for a new leader.”
- “Build a buddy/first-pair plan so this hire has belonging from day one.”

## What you get (deliverables)

1) Onboarding brief
2) Preboarding checklist
3) First-week plan + buddy/first-pair plan
4) Context pack outline
5) Listening tour kit (stakeholder map + questions + synthesis)
6) Working agreement (relationship design summary)
7) 30/60/90 + 1y/2y plan
8) 30-day state-of-the-union memo
9) Risks / Open questions / Next steps

Templates and checklists live in `skills/onboarding-new-hires/references/`.
