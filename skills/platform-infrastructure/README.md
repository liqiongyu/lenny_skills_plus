# platform-infrastructure

Produces a **Platform & Infrastructure Improvement Pack**: shared capabilities plan, quality targets (reliability/performance/privacy), scaling triggers, instrumentation decisions, and an execution roadmap.

## Install

This repo mirrors skills for both Codex and Claude Code. Canonical location:
- `skills/platform-infrastructure/`

Mirrors:
- `.codex/skills/platform-infrastructure/`
- `.claude/skills/platform-infrastructure/`

## Use

Invoke by asking for the deliverables explicitly. Example prompts:
- “Use `platform-infrastructure` to create a platformization plan for shared capabilities across teams. Include a doomsday clock and an execution roadmap.”
- “Use `platform-infrastructure` to define reliability/performance SLOs and the top infra projects for the next quarter.”
