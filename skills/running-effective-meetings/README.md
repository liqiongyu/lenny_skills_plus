# Running Effective Meetings (Skill Pack)

This skill helps a **team lead / product leader / facilitator** run effective meetings by producing concrete artifacts: meeting brief, pre-read, agenda, facilitation script, notes/decision log, and follow-up.

## Install

If you’re using this repo directly, the skill is discoverable under:
- `skills/running-effective-meetings`

Repo convention: this skill should also be mirrored to:
- `.codex/skills/running-effective-meetings`
- `.claude/skills/running-effective-meetings`

## Use

Invoke with a concrete request and context. Examples:
- “Create a decision meeting pack to pick a Q2 pricing change.”
- “Redesign our weekly ops meeting so it’s structured and stays out of strategy.”
- “Turn this calendar invite and context notes into a pre-read + agenda + facilitation script.”

Expected deliverable: a **Meeting Pack** as described in [SKILL.md](SKILL.md), using templates in [references/TEMPLATES.md](references/TEMPLATES.md).
