# Running Offsites (Skill Pack)

Creates a concrete **Offsite Pack**: brief, agenda, prework, facilitation run-of-show, logistics checklist, and post-offsite follow-through artifacts.

## Install

This repo mirrors skills for both Codex and Claude:
- Canonical: `skills/running-offsites/`
- Codex: `.codex/skills/running-offsites/`
- Claude: `.claude/skills/running-offsites/`

## Use

Invoke by name (“running offsites”) or by asking for common artifacts:
- offsite brief / offsite charter
- retreat agenda / run of show
- prework prompts / pre-read
- laptops-down whiteboard day plan
- quarterly burst plan

## What you’ll get

An **Offsite Pack** in this order:
1) Offsite Brief (1-pager)
2) Agenda + Session Output Map
3) Prework Pack
4) Facilitation Run-of-Show
5) Logistics Plan + Checklist
6) Post-Offsite Output Pack (decisions, action plan, comms, review)
7) Risks / Open questions / Next steps

Templates live in `references/TEMPLATES.md`.

## Example prompts

- “Plan a 1-day strategy offsite for my 12-person team. We need to pick Q2 priorities and leave with owners and dates. Output the full Offsite Pack.”
- “We’re remote-first. Create a quarterly 2-day burst program and a default burst agenda (connection + deep work).”
- “Draft an offsite run-of-show for a laptops-down whiteboard day. Include decision capture and a closing plan to prevent Monday amnesia.”
