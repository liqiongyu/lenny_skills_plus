# Technical Roadmaps (skill pack)

Creates a **Technical Roadmap Pack**: a written technical strategy (Rumelt), an execution-ready roadmap, initiative briefs, and an alignment cadence.

## What you get
- Technical Strategy (Diagnosis → Guiding policy → Coherent actions)
- Roadmap table (quarters or Now/Next/Later) with owners/deps/milestones/metrics
- Initiative briefs (top 3–6)
- Dependency + risk register
- Governance + comms plan
- Risks / Open questions / Next steps

## How to use
- Read: `skills/technical-roadmaps/SKILL.md`
- Copy/paste questions: `skills/technical-roadmaps/references/INTAKE.md`
- Use templates: `skills/technical-roadmaps/references/TEMPLATES.md`

## Example prompts
- “Create a 12-month technical roadmap for Platform Engineering. Audience is VP Eng + Product. Use quarterly format.”
- “Turn this list of tech-debt items into a prioritized technical roadmap with owners, dependencies, and metrics: <paste list>”

## Lint
